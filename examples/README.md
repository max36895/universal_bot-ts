# Примеры использования umbot

В этой директории содержатся примеры использования фреймворка `umbot` для различных сценариев.

## Структура примеров

- `skills/` - Базовые примеры использования
- `config/` - Настройки для приложения

## Как использовать примеры

1. Выберите интересующий вас пример
2. Скопируйте его в свой проект
3. Установите зависимости: `npm install`
4. Настройте конфигурацию (при необходимости)
5. Запустите: `npm start`

## Требования

- Node.js 20.0+
- TypeScript 5+
- umbot стабильная версия

## Дополнительная информация

Каждый пример содержит:

- README с описанием
- Полный исходный код
- Комментарии к коду
- Инструкции по настройке
- Примеры использования
