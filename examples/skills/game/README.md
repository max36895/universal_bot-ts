Игра Математика
=====================================================
Запуск
------------
1 Установить зависимости:
```bash
npm i
``` 
2 Соберите проект
```bash
npm run build
```
3 Запустить приложение:
```bash
npm start
```
4 Запустить nginx и полученную ссылку вставить в консоли разработчика навыка.
